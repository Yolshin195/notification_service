# Создание рассылки

## Описание

Этот эндпоинт предназначен для создания новой рассылки в системе. Позволяет отправить сообщение целевой аудитории с заданными параметрами.

## URL

`POST /api/v1/dispatches/`

## Параметры запроса

| Параметр                      | Тип           | Описание                                                  |
| ------------------------------ | ------------- | --------------------------------------------------------- |
| `timezone`                     | Строка        | Часовой пояс, в котором запланирована рассылка (например, "Africa/Abidjan"). |
| `start_datetime`               | Строка        | Дата и время начала рассылки в формате ISO 8601 (например, "2024-01-21T15:26:34.459Z"). |
| `end_datetime`                 | Строка        | Дата и время завершения рассылки в формате ISO 8601 (например, "2024-01-21T15:26:34.459Z"). |
| `message_text`                 | Строка        | Текст сообщения, который будет отправлен в рамках рассылки. |
| `filter_mobile_operator_code`  | Строка        | Код мобильного оператора для фильтрации аудитории (например, "3fa85f64-5717-4562-b3fc-2c963f66afa6"). |
| `filter_tag`                   | Строка        | Тег для дополнительной фильтрации аудитории (например, "3fa85f64-5717-4562-b3fc-2c963f66afa6"). |

## Пример запроса

```json
POST /api/v1/dispatches/
{
  "timezone": "Africa/Abidjan",
  "start_datetime": "2024-01-21T15:26:34.459Z",
  "end_datetime": "2024-01-21T15:26:34.459Z",
  "message_text": "Привет, мир!",
  "filter_mobile_operator_code": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "filter_tag": "3fa85f64-5717-4562-b3fc-2c963f66afa6"
}
```

## Ответ

В случае успешного создания рассылки, сервер вернет статус 201 Created и информацию о созданной рассылке в формате JSON. В противном случае, будет возвращен соответствующий статус ошибки с подробным описанием проблемы.

Пример успешного ответа:

```json
HTTP/1.1 201 Created
{
  "id": "df5b40b5-0f9e-4b14-9c05-4ff9f41e8cff",
  "status": "scheduled",
  "start_datetime": "2024-01-21T15:26:34.459Z",
  "end_datetime": "2024-01-21T15:26:34.459Z",
  "message_text": "Привет, мир!",
  "filter_mobile_operator_code": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "filter_tag": "3fa85f64-5717-4562-b3fc-2c963f66afa6"
}
```

## Ошибки

| Код ошибки                    | Описание                                                  |
| ------------------------------ | --------------------------------------------------------- |
| 400 Bad Request               | Некорректный запрос. Проверьте правильность параметров.    |
| 401 Unauthorized              | Недостаточно прав для выполнения операции.                 |
| 500 Internal Server Error     | Внутренняя ошибка сервера. Повторите попытку позже.       |
